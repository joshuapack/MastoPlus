# MastoPlus

Preview link: [https://www.joshuapack.pw/gplus](https://www.joshuapack.pw/gplus)

Start project.
```
npm i
npm run dev
```
then open [localhost:3000](http://localhost:3000) in the browser.

Build project.
```
npm run build
```
the files located in public folder are all you need
